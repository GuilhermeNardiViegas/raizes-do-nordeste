# raizes-do-nordeste

Projeto: Desenvolvimento Front-end (Raízes do Nordeste)

# o que foi implementado

 Fluxo de cadastro e autenticação com dados em mock.
 Seleção de unidade.
 Cardápio por unidade.
 Carrinho com total, desconto promocional, frete e pontos de fidelidade.
 Tela de perfil com controles de dados e privacidade.
 Painel administrativo com campanhas, pedidos e indicadores.
 Pagamento.
